# Text Analyzer (Python)

This project is a menu-based Python application that analyzes text files.
The program allows the user to perform different types of analysis on a text file through a simple and interactive command-line menu.

The project was developed as part of a Python programming course and focuses on practicing file handling, modular programming and user interaction.

## Features

- Count number of lines in a text file
- Count number of words
- Count number of letters
- Find the most frequently used words
- Find the most frequently used letters
- Analyze all metrics at once
- Switch between different input files
- Simple menu-based interface

## Project Structure

```bash
.
├── main.py        # Main program and menu logic
├── analyzer.py    # Functions for text analysis
└── *.txt          # Text files used for analysis
```

## How to Run

1. Make sure you have Python 3 installed.
2. Clone the repository or download the files.
3. Place a text file (for example phil.txt) in the project directory.
4. Run the program:

```bash
python main.py
```

5. Follow the on-screen menu instructions.
